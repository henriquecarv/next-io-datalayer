## System Requirements

- **[NodeJS](https://nodejs.org/en/)** (version >= 10).

## Installing

- `npm i next-io-datalayer`

## Usage

```javascript
import DispatchDataLayer from 'next-io-datalayer';

new DispatchDataLayer('generic-event-trigger', 'default action', 'default category', 'default label');
```
